# Pips & Lots

## Pip

A **Pip (Percentage in Point)** is the smallest standard price movement of a currency pair.

**Example:**

```text
EUR/USD
1.1050 → 1.1051 = 1 Pip
```

## Lot

A **Lot** is the standardized size of a trade in the Forex market.

It determines how much currency you are buying or selling.

### Common Lot Sizes

| Lot Type     |          Size |
| ------------ | ------------: |
| Standard Lot | 100,000 units |
| Mini Lot     |  10,000 units |
| Micro Lot    |   1,000 units |
| Nano Lot     |     100 units |

## Key Points

* A **Pip** measures price movement.
* A **Lot** measures trade size.
* Pip is used to calculate price changes.
* Lot is used to determine trading volume.

## Summary

A **Pip** tells you how much the price has moved, while a **Lot** tells you how large your trade is.
